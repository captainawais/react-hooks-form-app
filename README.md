# 🚀 React Hooks Form App

A modern React.js application developed to practice and understand:

- React Hooks
- useState
- useEffect
- Form Handling
- Event Handling
- Dynamic UI Rendering

This project was created as part of the Frontend Developer Internship Program at Appverse-Technologies.

---

# 📌 Project Description

The React Hooks Form App is an interactive web application that demonstrates how React handles dynamic state updates and user interaction using Hooks.

Users can:

- Enter their name and age
- See live updates instantly
- Experience dynamic UI rendering
- Submit and reset forms
- Observe real-time React state management

The project focuses on practical implementation of React fundamentals with a clean and responsive user interface.
