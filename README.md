# 🚀 React Hooks Form App

A modern React.js application developed to understand and practice:

- React Hooks
- useState
- useEffect
- Form Handling
- Dynamic UI Updates
- State Management

This project was created as part of my Frontend Developer Internship at Appverse-Technologies.

---

# 📌 Project Description

The React Hooks Form App is an interactive web application where users can:

- Enter their name and age
- Experience real-time form updates
- Learn how React state works
- Understand component re-rendering
- Practice event handling in React

The project demonstrates how modern React applications manage user data dynamically using Hooks.

Through this project, I learned:

- React component structure

- Side effects using `useEffect`

# ⚙️ Installation Instructions

Clone the repository:

```bash
git clone https://github.com/captainawais/react-hooks-form-app.git
```

Move into the project folder:

```bash
cd react-hooks-form-app
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm start
```

---

# 💡 Usage Guidelines

1. Enter your name
2. Enter your age
3. Click Submit
4. View live updates instantly
